# node-express-mongo
 Backend for react-ecommerce-shop

# Install 

```bash
npm init -y
npm install -g babel-cli
npm i --save nodemon express babel-cli babel-preset-env babel-preset-stage-0 dotenv
npm i body-parser --save
npm i mongoose --save
```
